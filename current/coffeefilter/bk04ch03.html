Chapter 3. Limitations

This chapter documents an assortment of implementation defined features and limitations.

  • The version of Unicode available depends on the version of Java used to build the library. This effects which character classes are recognized.

  • The ambiguity analyzer depends on its own Unicode support which does not include characters outside the basic multilingual plane. Also, character classes with many members tend to make it run more slowly.
